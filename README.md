# jmdict-idioms
JmdictFurigana からニ文字熟語や三文字熟語を機械的に生成します。

## Installation
1. install [JmdictFurigana](https://github.com/Doublevil/JmdictFurigana/releases)
2. ```npm install```

## Usage
```
node build.js
```
